Backgrounds and Borders

background-clip

背景の適用範囲を指定できる

  • background-clip:border-box; //(default)
  • background-clip:padding-box;
  • background-clip:content-box;

サンプル

background-clip:border-box;

background-clip:padding-box;

background-clip:content-box;

background-origin

背景の基準位置を指定

  • background-origin:padding-box; //(default)
  • background-origin:border-box;
  • background-origin:content-box;

サンプル

background-origin:padding-box;

background-origin:border-box;

background-origin:content-box;

background-size

背景画像のサイズを指定

  • background-size:auto;//(default)
  • background-size:contain;
  • background-size:cover;
  • background-size:●px ●px; //例）10px 10px
  • background-size:●% ●%;//例）50% 50%

サンプル

background-size:auto;

background-size:contain;

background-size:cover;

background-size:10px 10px;

background-size:50% 50%;

border-radius

ボックスの４つのコーナーの角丸の指定

  • border-radius: ●px;//４つの角全て同じサイズの角丸
  • border-radius: ●●px / ●●px ▼▼px ■■px ▲▲px;//右上省略(右上は、左上と同じ数値に)
  • border-radius: ●●px ▼▼px;//右下省略（右下は、左上と同じ数値に）
  • border-radius: ●●px ▼▼px ■■px;//左下省略（左下は、右上と同じ数値に）
  • border-radius: ●●px ▼▼px ■■px ▲▲px / ■■px ●●px ▲▲px ▼▼px;
    //４つの角全て水平方向・垂直方向のサイズが異なる角丸
    水平方向　左上・右上・右下・左下 / 垂直方向 左上・右上・右下・左下　で指定

サンプル

border-radius:４方向同じ大きさ
border-radius:20px;

border-radius:右上省略
border-radius:100px / 80px 320px 160px 240px;

border-radius:右下省略
border-radius:30px 140px;

border-radius:左下省略
border-radius:60px 100px 30px;

border-radius:４方向全て異なる
border-radius:40px 100px 30px 15px / 50px 30px 75px 65px;
